Добро пожаловать на ELSU Conference API !

Список страниц:

    /events
  • Список всех событий
  • Список всех событий постранично
  • Событие по {query:id}
  • Список событий со статусом {/status}
  • Список событий по рейтингу {/stars}

    • /members
  • Список участников
  • Участник по id

    • /statistics
  • Получить всю статистику
  • Получить статистику по событиям
  • Получить статистику по участникам

    • /uploads
  • Загрузить файл на сервер{post: file}
  • Получить файл с сервера {query:file(path)}

Статус событий:

  • 0 - Запланировано
  • 1 - идет
  • 2 - перенесено
  • 3 - отменено
  • 4 - завершено

Query для статистики

  • period - year, mount, week, day(current day) or date
  • question - status, platform, members
  • value - string, number (если есть у question)
  • filter - max, min
